    Data and method

    How the dashboard was built

    Unit of analysis

    The main analysis uses one row per state in the high-ST state master table. Variables combine ST education, poverty, labour, MGNREG, demographic, village-concentration, scholarship, and tribe-level support datasets.

    Database workflow

    Raw CSVs were cleaned into standardized outputs, merged into analysis-ready state tables, and then used to produce notebook EDA, correlation tables, and graph exports.

    Interpretation rule

    The dashboard reports associations, not causal estimates. Poverty coverage is limited, and some variables come from different years, so findings are policy signals rather than proof of cause.

    Policy reading

    What the evidence suggests

    Retention before expansion

    GER alone can be misleading. Secondary dropout is a better warning signal for poverty and livelihood distress, so policy should focus on transition and retention support.

    Education-livelihood mismatch

    Some states show reasonable schooling participation but continued MGNREG dependence or weak labour outcomes. These states need employment pathways, not only school access.

    Spatial targeting

    ST village concentration is most clearly linked with unmet MGNREG demand. Service delivery and livelihood programs should account for geographic concentration.
